# 可变参数列表
- 可变参数一定是方法中的最后一个参数
- 数组可以传递给可变参数的方法，反之不行
- 在重载中，含有可变参数的方法是最后被调用的 
# 方法的传值问题
- 基本数据类型传值 
  -   方法中对参数的修改，会对主方法中传来的数组产生影响 
 
- 引用数据类型 
  -  方法中对值的改变，会影响到主方法中传来的数组
